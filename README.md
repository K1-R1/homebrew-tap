# homebrew-tap

Homebrew formulae for [smoosh](https://github.com/K1-R1/smoosh).

## Install

```bash
brew install K1-R1/tap/smoosh

Available formulae

┌─────────┬──────────────────────────────────────────────────────────────────┐
│ Formula │                           Description                            │
├─────────┼──────────────────────────────────────────────────────────────────┤
│ smoosh  │ Pure bash CLI that aggregates repository files for RAG ingestion │
└─────────┴──────────────────────────────────────────────────────────────────┘

Troubleshooting

brew update
brew doctor

If the install fails, try brew untap K1-R1/tap && brew tap K1-R1/tap to refresh.
