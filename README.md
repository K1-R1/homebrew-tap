# homebrew-tap

Homebrew formulae for [smoosh](https://github.com/K1-R1/smoosh).

## Install

```bash
brew tap K1-R1/tap
brew install smoosh

Available formulae

┌─────────┬──────────────────────────────────────────────────────────────────┐
│ Formula │                           Description                            │
├─────────┼──────────────────────────────────────────────────────────────────┤
│ smoosh  │ Pure bash CLI that aggregates repository files for RAG ingestion │
└─────────┴──────────────────────────────────────────────────────────────────┘

Troubleshooting

brew update
brew doctor

If the install fails, refresh the tap:

brew untap K1-R1/tap && brew tap K1-R1/tap
